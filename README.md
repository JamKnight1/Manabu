# Project Title (Japanese Learning app)

A personal app that tests users on translating Japanese words into English

## Instructions for Build and Use

Steps to build and/or run the software:

1. Install Android studio 
2. Ensure either a virtual device is running or your own phone is connected to your device.
3. Run the app from Android Studio


Instructions for using the software:

1. Select one of the four buttons
2. If right, the app will show a correct screen (this has yet to be added)
3. If wrong, the app will show an incorrect screen and prompt you to return to the question


## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

*Android Studio
*A virtual or physical device to test with
*The kotlin libraries featured in the code
